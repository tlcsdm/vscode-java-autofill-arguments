# Change Log

## [1.0.0] - 2026-01-11

### Added
- Initial release
- Fill method arguments command with keyboard shortcut (Ctrl+Alt+O / Cmd+Alt+O)
- Context menu integration under tlcsdm submenu
- Code completion provider for argument suggestions
- Integration with Red Hat Java Language Support for better parameter resolution
- Configuration options for customizing behavior:
  - Enable/disable completion suggestions
  - Use parameter names or type names
  - Fallback behavior when parameter names unavailable
